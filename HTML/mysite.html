​

Motherboards are a vital component to computers, and one of the first pieces installed when assembling a new PC. Motherboards hold most of the crucial components in a computer and facilitate communication between them.

​

When purchasing a new motherboard there are many factors to consider.

 

First you must ensure that the board is compatible with the CPU (central processing unit) you own or plan to purchase. Different CPUs models are designed for different sockets. A socket is typically compatible with several CPUs, which allows you to upgrade your CPU without having to get a new board.  Manufacturers frequently release new sockets with modern CPUs, so you may have to upgrade your motherboard to keep up with the latest CPU models.

​

Each motherboards contains a number of memory slots and supports a certain memory standard. If you plan on using high memory applications (such as video rendering, photo editing, 3D modeling, intensive gaming, etc.) on your PC you might want to choose a motherboard with a high number of memory slots. Even if you don't use all slots initially, having more slots allows you to make RAM (Rapid Access Memory) upgrades in the future.

Memory standards (such as DDR3, DDR4, and DDR5) essentially describe how fast your RAM will operate with higher numbers indicating faster performance. Much like CPUs and their sockets, you should make sure the RAM you are using matches the same memory standard as your motherboard.

​

Each motherboard is designed to match a certain form factor. These form factors describe the dimensions of the board (and thus what PC cases it will fit into), where the screws will go, and what kind of power supply it will need. You should ensure that your motherboard is the right form factor for your case.
